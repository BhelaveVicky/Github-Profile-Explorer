# GitHub Profile Finder 🔍

A simple web application that allows users to search for GitHub profiles and view their details such as avatar, repositories, followers, and more.

## 🚀 Features

* Search any GitHub username
* Display user profile information
* Show avatar, bio, followers, and following
* List public repositories
* Clean and responsive UI

## ▶️ Usage

1. Enter a GitHub username in the search bar.
2. Click the search button.
3. The app will display the user's profile information.

Demo link :- https://github-profile-explorer-black.vercel.app/
